# Oyster Card

Week 2 Makers project.
Going more in depth with an emulation of the London Oyster card system.

## Instructions

Ruby version: 2.5.0

Install Ruby with RVM:

```
ruby -v # Current Ruby version
rvm install ruby-2.5.0
```

Install dependencies:

```
bundle install
```

dependencies: rspec

Feature test file:

```
ruby feature.rb
```
